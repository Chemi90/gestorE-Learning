# Quality Checklist

Base QA checklist for PR validation:

- [ ] Compila con Java 25
- [ ] Tests backend pasan
- [ ] Tests frontend pasan
- [ ] No hardcoded secrets
- [ ] Endpoints versionados `/api/v1`
- [ ] Errores devuelven JSON estructurado
- [ ] JWT validado correctamente
- [ ] Docker compose levanta correctamente
